  • datetime-year : 2011년 보다 2012년에 count가 높은 것을 보아 대여량이 증가하였으므로 성장했다고 판단 가능

  • datetime-month : 6월 ~ 10월 까지 대여량이 높으며, 12월 ~ 2월에는 대여량이 낮음을 확인

  • datetime-day : 1일 부터 19일 까지의 데이터만 존재함을 확인, 확인 결과 test 데이터에 이후의 일자가 있음을 확인

  • datetime-hour : 0시 부터 5시 까지는 count가 적음을 확인 할 수 있으며, 8시, 17시, 18시에 count가 높음을 확인

  • datetime-minute, second : 데이터가 없음(기록되지 않음)을 확인
